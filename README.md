# Analiza zmienności klimatu 

Projekt analizy danych klimatycznych dla obszaru wschodniej Sycylii w latach 2005–2024, obejmujący identyfikację trendów temperatury i opadów oraz wizualizację zmian w czasie.

## Technologie

Python, Pandas, NumPy, Matplotlib

## Przebieg pracy 

przetwarzanie i czyszczenie danych klimatycznych
analiza szeregów czasowych (temperatura, opady)
tworzenie wizualizacji zmian w czasie
interpretacja wyników i identyfikacja trendów

## Funkcjonalności

analiza danych klimatycznych z wielu lat
agregacja danych (np. miesięczna / roczna)
wizualizacja trendów temperatury i opadów
identyfikacja zmian klimatycznych

## Opis działania

Dane klimatyczne są analizowane jako szeregi czasowe. Projekt obejmuje ich przetwarzanie, agregację oraz wizualizację w celu identyfikacji długoterminowych trendów i zmian klimatycznych.

## Dane

Zbiór danych zawiera informacje o temperaturze oraz opadach dla wybranego regionu w okresie 2005–2024.
